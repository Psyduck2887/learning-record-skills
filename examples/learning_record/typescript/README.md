# TypeScript 学习记录

## 读取规则

读取这个主题时，加载：

1. `notes.md`

读取完成后，输出：

```text
已读取 TypeScript 学习记录。
```

## 写入规则

目标文件：

```text
notes.md
```

每条新记录使用二级标题开头：

```md
## 简短笔记标题

对已经解决的问题做紧凑说明。
```

每条笔记只记录一个已经解决的问题。只有在确实更容易复用时，才使用列表、命令或代码块。

追加前，先检查 `notes.md` 是否已经包含类似笔记。如果已有类似内容，不要重复追加。

这个示例使用一个空行分隔不同笔记。你可以在自己的主题 `README.md` 里定义其他间距规则。

## 验证

这个示例主题没有验证命令。如果你的主题需要验证，请在这里写明具体命令。
